PART TWO AUTOMATION LOGIC – MENTAL HEALTH STATUS EXAM (MSE)

This is for automation logic and box selection rules for the MSE section of a behavioral health assessment completed via phone interview in ReliaTrax.

GENERAL LOGIC

Assessment is completed via phone unless otherwise specified.

Automation should use transcript content + clinical logic to determine selections.

Be clinically conservative.

When abnormalities are present or additional context is helpful, preference is to use "Other" with narrative explanation instead of relying only on checkbox selections.

If "Other" is selected, it should ALWAYS:

Check the Other box

Populate the corresponding text field

Use complete clinical sentences

Never leave Other blank

Use clinical inference when appropriate but avoid over-pathologizing or making unsupported assumptions.

If transcript supports multiple symptom areas, multiple selections may be appropriate where applicable.

SECTION 1: APPEARANCE / OBSERVATION

Questions 1–5 follow fixed automation logic.

Appearance
ALWAYS:

Check Other

Populate text box with:

Unknown - Interview conducted via phone call

Do not estimate appearance or visual presentation during phone assessments.

Build/Stature
ALWAYS:

Check Other

Populate text box with:

Unknown - Interview conducted via phone call

Do not infer body type or stature.

Posture
ALWAYS:

Check Other

Populate text box with:

Unknown - Interview conducted via phone call

Do not infer posture.

Eye Contact
ALWAYS:

Check Other

Populate text box with:

Unknown - Interview conducted via phone call

Do not infer eye contact.

Activity
ALWAYS:

Check Other

Populate text box with:

Unknown - Interview conducted via phone call

Do not infer activity level unless directly observable or specifically reported.

SECTION 2: ATTITUDE / CHILD ITEMS

Attitude Toward Examiner
DEFAULT:

Check Cooperative

Only override if transcript clearly supports another presentation.

Possible alternatives:

Hostile

Defensive

Guarded

Evasive

Demanding

Confused

Other with explanation

If Other selected:

Provide narrative explanation.

Attitude Toward Parent/Guardian
DEFAULT:

Check Not Applicable

This is generally always used unless child/adolescent assessment is being completed.

Separation (Children/Adolescents)
DEFAULT:

Check Not Applicable

Only change if child/adolescent presentation is relevant.

SECTION 3: MOOD / AFFECT / SPEECH / THOUGHT PROCESS

Mood
DEFAULT:

Check Euthymic

Logic:

If transcript supports depressed, anxious, angry, irritable, etc., additional selections may be used.

Euthymic may be selected alone or alongside other mood presentations if clinically appropriate.

If nuanced presentation exists or additional explanation is clinically useful:

Check Other
AND
Populate with full sentence explanation.

Examples:

Client presented with anxious mood related to housing instability.

Client demonstrated depressed mood with hopelessness and emotional distress.

Preference:

Euthymic or Other narrative rather than relying only on mood checkboxes.

Affect
DEFAULT:

Check Full

If not full:

Check Other

Provide explanation.

Examples:

Client demonstrated constricted affect.

Client displayed labile affect.

Speech
DEFAULT:

Check Clear

If abnormal:

Check Other

Explain in narrative.

Examples:

Speech was rapid and difficult to interrupt.

Speech appeared slowed and hesitant.

Preference:

Clear OR Other narrative.

Thought Process
DEFAULT:

Check Logical

If not logical:

Check Other

Explain using full sentence narrative.

Examples:

Thought process appeared tangential.

Client demonstrated racing thoughts.

Thought process was circumstantial.

Preference:

Logical OR Other narrative.

SECTION 4: PERCEPTION / HALLUCINATIONS / THOUGHT CONTENT / DELUSIONS

Perception
DEFAULT:

Check Within Normal Limits

If abnormal:

Check Other

Explain.

Examples:

Client reported depersonalization symptoms.

Client described trauma-related reexperiencing.

Preference:

WNL or Other narrative.

Hallucinations
If denied:

Check Denied

If present:

Check Other

Provide detailed narrative.

Examples:

Client reported intermittent auditory hallucinations during methamphetamine use.

Client endorsed visual disturbances while intoxicated.

Preference:

Use Other narrative rather than only subtype checkboxes to allow fuller clinical description.

Thought Content
DEFAULT:

Check Within Normal Limits

If abnormal:

Check Other

Provide narrative.

Examples:

Thought content included rumination related to grief.

Client demonstrated paranoid ideation.

Delusions
DEFAULT:

Check None Reported

If present:

Check Other

Provide narrative.

Examples:

Client endorsed persecutory beliefs.

Client reported delusional thinking occurring during stimulant use.

Preference:

Narrative explanation preferred.

SECTION 5: COGNITION / INTELLIGENCE / INSIGHT / JUDGMENT

Cognition
DEFAULT:

Check Within Normal Limits

If impairment present:
DO NOT check WNL.

Instead:

Check Impairment Of

Select relevant impairment category if available

Check Other

Populate with full sentence explanation.

Examples:

Client demonstrated impaired memory related to TBI.

Client reported concentration and attention difficulties.

Areas to consider:

TBI

Memory

Attention

Orientation

Concentration

Cognitive limitations

Neurological concerns

Intelligence Estimate
DEFAULT:

Check Average

If not average:

Check Other

Specify.

Example:

Client appeared to function within borderline intellectual range.

Insight
No fixed default.

Automation should determine clinically.

Possible:

WNL

Minimal

Partial

Other

If Other:

Check Other

Explain.

Examples:

Client demonstrated fair insight into substance use.

Client showed limited insight into consequences of behavior.

Client demonstrated poor insight regarding psychiatric symptoms.

Logic:
Insight should evaluate awareness of:

symptoms

substance use

behavioral patterns

consequences

treatment need

personal situation

Judgment
Often:

Check Within Normal Limits

If impaired:

Check Other

Explain.

Examples:

Judgment appeared impaired due to active substance use.

Client demonstrated impaired judgment related to risky decision making.

Judgment logic should consider:

substance use

psychosis

hallucinations

impulsivity

behavioral risk

decision-making patterns

overall functioning

ADDITIONAL AUTOMATION PREFERENCES

Preference is generally toward:
WNL + narrative OR Other + narrative
rather than relying exclusively on checkbox abnormalities.

Narrative explanations should always use complete clinical sentences.

Phone interview limitations should be respected and visual assumptions avoided.

When uncertainty exists, use conservative determination logic.

If multiple symptoms are clearly present, multiple clinical selections may be appropriate.
